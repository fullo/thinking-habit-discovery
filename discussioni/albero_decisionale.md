# Albero Decisionale: Quick Profile (3-5 minuti)

## Istruzioni

Questo strumento ti dà un profilo rapido in 8-12 domande. Parti dalla domanda 1 e segui le frecce in base alla tua risposta. Ogni ramo ti porta a domande diverse, costruendo il tuo profilo passo dopo passo.

A differenza del questionario completo, qui ogni domanda determina la successiva. Il risultato è meno granulare ma molto più veloce.

---

## RAMO 1 — Canale cognitivo (D1)

### Domanda 1: Il primo terreno

> Stai sotto la doccia e ti viene un'idea brillante. In che forma esiste quell'idea nella tua testa, nell'istante prima di fare qualsiasi cosa?

- **"Me la sto dicendo in testa, a parole"** → **D1 = Verbale (1.1)** → vai a Domanda 2A
- **"La vedo: è un'immagine, uno schema, una scena"** → **D1 = Visuo-spaziale (1.2)** → vai a Domanda 2B
- **"La sento: un'eccitazione, una certezza fisica, un'energia"** → **D1 = Somatico (1.3)** → vai a Domanda 2C
- **"Non ha forma precisa — so che è giusta ma non ha né parole né immagini"** → vai a Domanda 1bis

### Domanda 1bis: Disambiguazione

> Quando dici "non ha forma precisa" — è più vicino a:

- **"Un senso di equilibrio, come quando una melodia risolve"** → **D1 = Sonoro/ritmico (1.4)** → vai a Domanda 2C
- **"Un sapere astratto, puro, come un concetto matematico"** → **D1 = Astratto-proposizionale (1.5)** → vai a Domanda 2B

---

## RAMO 2 — Strategia dominante (D2)

### Domanda 2A (per Verbali)

> Quell'idea che ti sei detto nella testa — ora cosa ne fai?

- **"La racconto a qualcuno per farla attaccare e vedere se regge"** → **D2 = Adversariale (2.1)** → vai a Domanda 3
- **"Mi faccio domande: perché la penso? Cosa assumo? Cosa mi sfugge?"** → **D2 = Socratico (2.2)** → vai a Domanda 3
- **"La spiego a qualcuno (o a me stesso) come se dovessi insegnarla"** → **D2 = Insegnamento (2.4)** → vai a Domanda 3
- **"La scrivo, poi la rileggo, poi la riscrivo meglio"** → **D2 = Iterativo (2.3)** → vai a Domanda 3

### Domanda 2B (per Visuo-spaziali / Astratti)

> Quell'immagine o quel concetto — come lo fai evolvere?

- **"Lo smonto in pezzi per capire come funziona"** → **D2 = Decostruttivo (2.5)** → vai a Domanda 3
- **"Cerco a cosa somiglia — qualcosa che ho già visto in un altro contesto"** → **D2 = Analogia (2.8) o Pattern recognition (2.14)** → vai a Domanda 3
- **"Lo traduco in un altro linguaggio: codice, formula, schema"** → **D2 = Traduzione tra domini (2.11)** → vai a Domanda 3
- **"Genero varianti: e se fosse così? O così? O così?"** → **D2 = Generativo (2.10)** → vai a Domanda 3

### Domanda 2C (per Somatici / Sonori)

> Quella sensazione — come la trasformi in qualcosa di utilizzabile?

- **"La metto in pratica subito: provo, costruisco, testo"** → **D2 = Iterativo (2.3)** → vai a Domanda 3
- **"Aspetto — accumulo altre sensazioni/informazioni finché non scatta"** → **D2 = Accumulo (2.6)** → vai a Domanda 3
- **"Cerco di metterla in parole o in uno schema, con fatica"** → **D2 = Insegnamento/auto-spiegazione (2.4)** → vai a Domanda 3
- **"La confronto con le mie esperienze passate: quando ho sentito qualcosa di simile?"** → **D2 = Pattern recognition (2.14)** → vai a Domanda 3

---

## RAMO 3 — Direzione (D3)

### Domanda 3: Dove vai

> Stai affrontando un problema che ha troppe variabili. Il tuo istinto è:

- **"Esplorare tutto: generare opzioni, allargare il campo"** → **D3 = Divergente (3.1)** → vai a Domanda 4
- **"Tagliare subito: eliminare il rumore e andare dritto al nocciolo"** → **D3 = Convergente (3.2)** → vai a Domanda 4
- **"Ribaltarlo: guardarlo da un'angolazione completamente diversa"** → **D3 = Laterale (3.3)** → vai a Domanda 4
- **"Semplificarlo: togliere variabili finché non diventa gestibile"** → **D3 = Sottrattivo (3.6)** → vai a Domanda 4

### Domanda 3bis (opzionale, per raffinare D3):

> Quando trovi una buona soluzione, tendi a:

- **"Cercare il principio generale dietro: cosa implica a livello più ampio?"** → aggiorna D3 con **Astrattivo (3.4)**
- **"Testarla su un caso specifico: funziona davvero nella pratica?"** → aggiorna D3 con **Concretizzante (3.5)**
- **"Comprimerla: come la esprimo in una frase?"** → aggiorna D3 con **Compressivo (3.7)**
- **"Proiettarla nel tempo: funzionerà ancora tra 5 anni?"** → aggiunta D3 con **Scala temporale (3.8)**

---

## RAMO 4 — Medium (D4)

### Domanda 4: Come esce

> L'idea ha preso forma. Ora devi lavorarci concretamente. La prima cosa che fai è:

- **"Ne parlo con qualcuno, o ad alta voce da solo"** → **D4 = Voce (4.1)**
- **"Apro un file/quaderno e comincio a scrivere"** → **D4 = Scrittura (4.2)**
- **"Prendo carta e disegno uno schema"** → **D4 = Diagramma (4.3)**
- **"Costruisco un prototipo o modello, anche grezzo"** → **D4 = Gesto/prototipo (4.4)**
- **"Scrivo codice, formule, o specifiche formali"** → **D4 = Codice/formalizzazione (4.5)**

### Domanda 4bis: Per chi esce

> Quella cosa che hai prodotto (testo, schema, prototipo, codice) — a chi serve prima di tutto?

- **"A me: mi serve per capire meglio"** → **Dest. = sé**
- **"A qualcun altro: lo condividerò per comunicare"** → **Dest. = altro umano**
- **"Al me del futuro: lo rileggerò/riuserò dopo"** → **Dest. = sé futuro**

---

## RAMO 5 — Forma logica (D5)

### Domanda 5: Come concludi

> Hai lavorato sull'idea, l'hai fatta evolvere. Quando dici "sono arrivato alla conclusione", come ci sei arrivato?

- **"Logicamente: se X allora Y, necessariamente"** → **D5 = Deduttivo (5.1)**
- **"Per esperienza: le ultime N volte che ho visto qualcosa di simile, è andata così"** → **D5 = Induttivo (5.2)**
- **"Per la spiegazione più plausibile: non ne sono certo, ma è la migliore ipotesi"** → **D5 = Abduttivo (5.3)**
- **"Per analogia: funziona come in quell'altro caso, quindi..."** → **D5 = Analogico (5.4)**

## COMPOSIZIONE DEL PROFILO

Alla fine dell'albero hai 5 valori (uno per dimensione). Il tuo profilo rapido è:

```
┌──────────────────────────────────────────────────────────────┐
│  PROFILO RAPIDO DI PENSIERO                                  │
│                                                              │
│  Canale:    _____________ (D1)                               │
│  Strategia: _____________ (D2)                               │
│  Direzione: _____________ (D3)                               │
│  Medium:    _____________ (D4) → per: _________              │
│  Logica:    _____________ (D5)                               │
│                                                              │
│  Gap potenziale: D1 _____ vs D4 _____                        │
│  (se diversi: attenzione alla perdita informativa)           │
└──────────────────────────────────────────────────────────────┘
```

**Nota sul gap D1↔D4:** Se il tuo canale cognitivo e il tuo medium di esternalizzazione sono diversi (es. pensi in immagini ma esternalizzi a parole), c'è un passaggio di traduzione dove puoi perdere informazione. Non è un difetto — è un punto di consapevolezza.

Per un'analisi più approfondita, usa il **questionario completo** (30 domande) oppure consulta la **guida interpretativa** (`interpretazione_profili.md`).
