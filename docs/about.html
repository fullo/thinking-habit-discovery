Questo progetto nasce da una domanda: come pensiamo, davvero? Non quanto siamo bravi a pensare, ma quale forma prende il nostro pensiero.

Il modello a 5 dimensioni

Abbiamo identificato 5 dimensioni indipendenti che descrivono il profilo di pensiero di una persona:

Claude Code Skill

Questo modello e disponibile anche come plugin per Claude Code. Una volta installato, Claude adatta automaticamente le sue risposte al tuo profilo cognitivo.

/thinking-profile

Scopri il tuo profilo

15 domande scenario-based + approfondimento adattivo. Salva il profilo in memoria: Claude si adatta a come pensi in tutte le conversazioni future.

/thinking-translator

Traduci i tuoi prompt

Scrivi nel tuo modo naturale. La skill riformula il prompt nel formato ottimale per il LLM, colmando il gap tra come pensi e come il LLM riceve.

/thinking-bridge

Media tra profili

Prendi un messaggio scritto da te e riformulalo per qualcuno con un profilo cognitivo diverso. Utile per comunicazione cross-team.

Installazione

claude plugin marketplace add fullo/claude-plugins-marketplace
claude plugin install thinking-habit@fullo-plugins

Richiede Claude Code CLI. Il profilo viene salvato in memoria e persiste tra sessioni.
